As a user, I can quit the game


A user can quit the game using the close button. This will prompt whether they want to save the game before fully exiting the game.


Acceptance criteria:
* User uses the red X at the top of the GUI to quit the game
* User gets a prompt to save the game
* Application closes entirely

* Unit testing
* Write testcase
